# MicroBench

Overview of our data for microbial genomic benchmarking. We have
sequenced a bunch of mono cultures, and metagenomic samples. Data has
been deposited to the ENA
([PRJEB85558](https://www.ebi.ac.uk/ena/browser/view/PRJEB85558)).

Monocultures:

- [Anabaena variabilis PCC 7120 DSM
  107007](#anabaena-variabilis-pcc-7120-dsm-107007)

  11109](#desulfobacca-acetoxidans-asrb2-dsm-11109)
- [Sagittula stellata E-37 DSM
  11524](#sagittula-stellata-e-37-dsm-11524)

  13772](#salmonella-bongori-1224.72-dsm-13772)
